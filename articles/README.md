# 文章目录

把你的原创文章放在这里。

## 命名规则

文件名格式：`YYYY-MM-DD-文章标题.md`

例如：
- `2026-07-20-萝卜快跑爆火36岁失业中年人怎么看.md`
- `2026-07-21-为什么中年失业后我反而更清醒了.md`

## 发布步骤

1. 点击仓库主页的 `Add file` → `Create new file`
2. 在路径框输入：`articles/2026-07-20-你的文章标题.md`
3. 把我写好的文章内容粘贴进去
4. 点 `Commit changes`
5. 等 1-2 分钟，网站自动更新
